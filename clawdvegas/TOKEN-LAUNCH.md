## How Clanker Works

Clanker is an autonomous token deployer on Farcaster. To launch:

1. Post on Farcaster/Warpcast mentioning @clanker
2. Include token name, ticker, and image
3. Clanker deploys the token on Base automatically
4. Liquidity is created via Uniswap
5. Clanker takes a small fee that goes to $CLANKER buybacks
